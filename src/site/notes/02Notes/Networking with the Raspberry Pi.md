# What is Networking?

**Networking** is the practice of connecting computers and other devices so they can share data, resources, and services.

## Networking methods for raspberry pi

There are a variety of methos for connecting to a raspberry pi including

1. Wi-fi 
2. Bluetooth
3. Web Server 
4. SSH
